1. NEW FEATURES SUMMARY
================================================================================

1.1 Tile Mesh Edit System (P0 — Core)
    - 5 edit presets: CLEAN_NOISE, DECIMATE_TO_TARGET, GENERATE_LODS,
      GENERATE_COLLIDER_PROXY, PACK_FOR_UNITY
    - Async job queue with SQLite persistence
    - Blender headless worker (subprocess → tile_edit.py)
    - Preview → Apply 2-step workflow
    - Edit layer versioning (original tiles READ-ONLY)
    - Tile boundary vertex preservation during decimation
    - Seam risk measurement (boundary vertex displacement detection)
    - Rollback to any version or revert to raw
    - Version comparison (before/after stats)
    - Tile validation (missing textures, naming, size)
    - Quality report generation
    - NLU integration (Korean + English natural language commands)
    - WebSocket progress broadcasting (5 event types)

1.2 Project/Data Wizard (P1)
    - OBJ folder scanning + validation
    - Tile naming convention check
    - Missing texture/MTL detection
    - Polygon/vertex/size statistics
    - Memory estimation
    - Recommended preset (performance/balanced/quality)
    - Preset guide with use cases

1.3 Bookmarks / Work Sets (P1)
    - Save camera position + target + zoom
    - Category filtering (general, measurement, review)
    - Instant camera restore in 3D viewer
    - SQLite persistence
    - CRUD REST API

1.4 Annotations & Snapshot Export (P1)
    - 3D pin placement mode (click-to-place markers on scene)
    - Pin + memo on 3D view with hover tooltips
    - Server-side persistence (via bookmarks API, "annotations" category)
    - Load annotations from server on page load
    - PNG export of current 3D viewport
    - PDF/HTML report export (printable annotation summary)
    - Annotation list in UI

1.5 Performance Auto-Tuner (P1)
    - Real-time FPS / Memory / Draw Calls / Triangles monitoring
    - Color-coded meters (green/yellow/red)
    - Auto-recommendations when thresholds exceeded
    - Backend metrics integration (tile report → polygon stats)
    - Auto-preset suggestion (clean_noise / decimate based on metrics)
    - One-click preset apply from suggestions
    - Start/Stop toggle with 30s backend polling

1.6 Tool Mode Panel (P1)
    - Tabbed panel: Measure / Path / Visibility
    - Measure: distance/angle measurement with results list
    - Path: start/end point selection + A* pathfinding + distance calc
    - Visibility: sensor placement + viewshed computation + report export
    - Toggle via scene toolbar wrench button

1.7 Unity Integration (P0)
    - TileEditApplier: Scan → Import → SwapLOD → SwapCollider
    - CityTileStreamer: active_versions.json awareness, auto-reload
    - NavMesh rebuild menu item (Vibe3D > Rebuild NavMesh)
    - Edit layer hierarchy setup (BaseLayer / EditLayer)
    - Gizmo differentiation (orange = edited tiles)

10. DATA/VERSION MANAGEMENT
================================================================================

/tiles_raw/                          Original OBJ (READ-ONLY)
/tiles_work/                         Job temporary workspace
/tiles_edit/
  /tile_{x}_{y}/
    /v0001/
      tile_LOD0.fbx
      tile_LOD1.fbx
      tile_LOD2.fbx
      tile_COLLIDER.fbx
      manifest.json
    /v0002/ ...
  active_versions.json               {tile_id: active_version}

manifest.json contains:
  - tile_id, version, preset, params
  - input_stats (faces, vertices, size)
  - output_stats (LOD0/1/2 faces, collider faces)
  - quality_flags (seam_risk, warnings)
  - timing, created_at

11. NLU INTEGRATION
================================================================================

Natural language commands (Korean/English) automatically detected:
  "tile_0012_0007 타일을 60만 폴리곤으로 경량화해줘"
  "decimate tile_0012_0007 to 600k"
  "타일 정리" → CLEAN_NOISE
  "LOD 생성" → GENERATE_LODS
  "콜라이더 재생성" → GENERATE_COLLIDER_PROXY

Pipeline: NLU → plan_generator (template match) → plan_validator
  → executor (_mesh_edit_tile handler) → mesh_edit_manager.start_job()
